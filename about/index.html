About TechTinkering

TechTinkering is devoted to Retro Computers, Programming and General Technical Tinkering. Its aim is to produce clear, easy to follow articles so that you can learn from, extend and recreate yourself. The main author is Lawrence Woodman.

TechTinkering uses jekyll to create a statically generated site and the source for the site is hosted on GitHub.

Guest Authors

If you would like to write an article for the site, then please drop me an email or send a pull request. I would love to hear from you.
